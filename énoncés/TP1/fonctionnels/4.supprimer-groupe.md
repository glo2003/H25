# Supprimer un groupe

En tant que locataire, je peux supprimer un groupe.

## Critères de succès

- Le groupe est supprimé
- Même si le nom de groupe n'est pas trouvé, la requête retourne un succès

## Spécifications

### Requête

**Path**

`DELETE /groups/{groupName}`

### Réponses

- `204 No Content`: Succès
